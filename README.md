# Oasis Infobyte Web Development Internship (OIBSIP)

This repository contains all the completed tasks for Level 1 of the Web Development & Designing internship track.

## 📂 Project Structure
* **WebDev-L1-LandingPage** - A responsive, static product landing page built using modern CSS layout techniques (Flexbox/Grid).
* **WevDev-L1-Portfolio** - A personal portfolio website showcasing skills, projects, and contact info with smooth scroll behaviors.
* **WebDev-L1-TemperatureConverter** - An interactive utility that dynamically converts temperature inputs between Celsius, Fahrenheit, and Kelvin scales.

## 🛠️ Tech Stack Used
* HTML5
* CSS3
* JavaScript (Vanilla)
